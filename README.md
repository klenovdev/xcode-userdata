# xcode-userdata

Installation
---- 

### Using the installation script

    $ git clone https://github.com/klenovdev/xcode-userdata.git
    $ cd xcode-userdata
    $ ./install.sh # And then restart Xcode

### Alternative way

    $ git clone https://github.com/klenovdev/xcode-userdata.git
    $ cd xcode-userdata
    $ mkdir -p ~/Library/Developer/Xcode/UserData/
    $ cp -R * ~/Library/Developer/Xcode/UserData/
